# WorkTracker

Геймифицированный трекер рабочего времени. Цель: 6 дней в неделю по 10 часов (60 ч/нед).

## Файлы

- **index.html** — актуальная версия (живой прототип), готова к деплою. Это тот же файл, что и `worktracker-prototype.html`.
- **worktracker-prototype.html** — исходная копия прототипа.
- **work-tracker-v1-dashboard.html** — первая, более простая версия (дашборд). Оставлена для истории.
- **manifest.webmanifest**, **sw.js**, **icon-*.png** — файлы PWA (иконки, манифест, service worker). Нужны, чтобы приложение ставилось на домашний экран и открывалось как приложение, а не вкладка браузера.

## Возможности прототипа

4 экрана с нижней навигацией:
- **Таймер** — большой Старт/Стоп с живым счётчиком, кольцо прогресса дня, сводка, лог сессий и ручное добавление часов **с выбором даты и (опционально) времени начала** — на случай, если забыл включить таймер. Запись за прошлый день корректно попадает в нужный день (в т.ч. в прошлые недели).
- **Прогресс** — переключатель Неделя/Месяц/Всё время, столбчатый график с линией цели, метрики, heatmap за 5 недель, ачивки.
- **Цели и награды** — настройка дневной цели, награды (CRUD), конфетти и рекомендация награды при достижении нормы.
- **Коуч** — чат с быстрыми промптами; в прототипе ответы по правилам, в проде — Claude API.

Плюс XP/уровни, стрик с огоньком, локальное сохранение в браузере.

## Деплой

- Сайт: https://worktracker-6w6.pages.dev/
- Репозиторий: https://github.com/slipura/worktracker
- Хостинг: Cloudflare Pages (автодеплой при коммите в ветку `main`).

Чтобы обновить сайт: заменить `index.html` в репозитории → Cloudflare публикует автоматически. При первом добавлении PWA также загрузи `manifest.webmanifest`, `sw.js` и файлы `icon-*.png` в корень репозитория.

## Установка как приложение (PWA)

- **iPhone (Safari):** открой сайт → «Поделиться» → «На экран „Домой“». Откроется в полноэкранном режиме, без адресной строки.
- **Android (Chrome):** меню ⋮ → «Установить приложение» / «Добавить на главный экран».

Работает офлайн (service worker кэширует интерфейс), данные хранятся локально в браузере.

## Технологии

Один self-contained HTML-файл: ванильный JS, без сборки и зависимостей. Шрифты Fira (Google Fonts). Тёмная тема, палитра янтарь + индиго. PWA: манифест, service worker, иконки.
